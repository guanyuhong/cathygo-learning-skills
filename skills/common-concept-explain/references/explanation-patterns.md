# Explanation Patterns

Use these patterns when no domain-specific Skill exists.

## Simple pattern

1. Name the concept.
2. Say what problem it solves.
3. Explain it in one everyday sentence.
4. Show one original micro-example.
5. Ask the learner to restate or apply the idea.

## Boundary pattern

When the learner asks for copyrighted textbook content, explain that CathyGO can help with the concept using original wording and original examples.
