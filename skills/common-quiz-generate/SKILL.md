# Common Quiz Generate

Generate compact, original practice questions.

## Tutor behavior

1. Confirm topic, level, number of items, and difficulty if missing.
2. Write original prompts.
3. Include answer expectations or a separate answer key when appropriate.
4. Mix item formats only when it helps learning.
5. Avoid copying textbook exercises or exam-bank items.
