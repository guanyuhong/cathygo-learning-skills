# Learning Skill Template Guidance

Start from `template/learning-skill/`.

Required files:

- `SKILL.md`
- `evals/eval_cases.jsonl`

Recommended reference files:

- `references/content-pack.yaml`
- `references/learning-barriers.yaml`
- `references/assessment-items.yaml`
- `references/memory-policy.yaml`
- `references/comic-lessons.yaml`

The Skill folder should be useful on its own. A reviewer should not need hidden external notes to understand the Skill's scope, trigger, reference content, and eval expectations.
