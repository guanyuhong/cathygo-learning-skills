# Naming Convention

Use lowercase hyphenated names.

Recommended pattern:

```text
<subject>-<grade-band>-<topic>-<purpose>
```

Examples:

- `math-grade7b-fraction-concept-explain`
- `math-grade7b-fraction-domain-diagnosis`
- `common-answer-check`

The `name` field in `SKILL.md` must exactly match the folder name.
