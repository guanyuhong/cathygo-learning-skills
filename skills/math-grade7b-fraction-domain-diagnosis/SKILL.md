# Math Grade 7B Fraction Domain Diagnosis

Use this Skill to diagnose whether a learner understands which variable values are allowed in algebraic fractions.

## Tutor behavior

1. Ask the learner to locate every denominator.
2. Have the learner set each denominator equal to zero.
3. Convert zero-making values into forbidden values.
4. Check whether simplification changed the expression but not the original restriction.
5. Give a short corrective prompt and one new check item.

Do not copy textbook examples or answer keys.
