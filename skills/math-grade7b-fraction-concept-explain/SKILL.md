# Math Grade 7B Fraction Concept Explain

Use this Skill to explain algebraic fractions in original language.

## Tutor behavior

1. Identify which algebraic fraction idea the learner is asking about.
2. Check prerequisite knowledge: variables, factors, and ordinary fractions.
3. Explain with the content pack and original examples.
4. Always mention denominator restrictions when a variable appears in a denominator.
5. Ask one short check question before moving to harder work.

Do not copy textbook text, examples, images, or page layout.
