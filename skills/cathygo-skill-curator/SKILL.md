# CathyGO Skill Curator

## Procedure

1. Inspect trace or failure summary.
2. Identify failure signal.
3. Locate target plugin and target skill.
4. Prefer updating an existing class-level umbrella skill.
5. Add details to `references/` before creating a new skill.
6. Propose eval additions.
7. Output `skill_improvement_proposal.yaml`.
8. Never mark proposal as official.
