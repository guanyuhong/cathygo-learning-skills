# Answer Checking Rubric

Use this rubric when no domain-specific checker exists.

## Correct

- Confirm correctness.
- Name the reasoning step that made the answer valid.
- Offer one optional extension.

## Partly correct

- Identify the valid part.
- Point to the first issue.
- Ask the learner to revise.

## Incorrect

- State that the answer needs revision.
- Give one targeted hint.
- Avoid shaming language.

## Not enough information

Ask for the original question or the learner's working.
