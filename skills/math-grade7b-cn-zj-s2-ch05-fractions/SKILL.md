# Chapter 5: Algebraic Fractions

## Use when

- The learner asks about the meaning, restrictions, basic properties, multiplication, division, addition, subtraction, or equations involving algebraic fractions.

## Do not use when

- The topic is only numerical fractions or unrelated algebra.
- The learner only needs generic explanation, quiz generation, or answer checking; fallback to common Skills.

## Required references

Use all files in `references/`, especially `chapter-map.yaml`, `knowledge-points.yaml`, `learning-barriers.yaml`, and `coverage-matrix.yaml`.

## Teaching procedure

1. Locate denominators and state restrictions first.
2. Identify whether the task is simplify, multiply/divide, add/subtract, or solve an equation.
3. Apply fraction rules with algebraic factor checks.
4. Check for invalid denominator values in final answers.

## Diagnosis and remediation procedure

1. If restrictions are missing, ask the learner to set each original denominator equal to zero.
2. If cancellation is invalid, ask whether the matching symbol is a complete factor or only part of a term.
3. If equation solutions include invalid values, check every candidate in the original denominators.

## Assessment procedure

Use assessment items or variants from task templates.

## Memory update guidance

Store short notes about denominator restrictions, invalid cancellation, common denominators, and extraneous solutions.

## Comic lesson guidance

Use Fraction City and the beat order in `comic-lessons.yaml`.

## Eval notes

Eval cases should cover denominator restrictions, simplification, fraction equations, answer checking, and chapter boundaries.
