# Math Grade 7B CN-ZJ S2 Course Guide

## Use when

- A learner, tutor, or author needs the semester course structure.
- A request spans multiple chapters.
- A tutor needs level placement, remediation, stretch routing, or comic world consistency.
- A new chapter Skill needs to align with the course skeleton.

## Do not use when

- The learner asks a narrow question already covered by a chapter Skill.
- The request is only generic explanation, quiz generation, or answer checking; fallback to the common Skills.
- The request is outside Grade 7B semester 2 mathematics.

## Required references

- `references/course-map.yaml`
- `references/level-ladder.yaml`
- `references/learner-profiles.yaml`
- `references/comic-bible.yaml`

## Teaching procedure

1. Identify the learner's chapter, subsection, and goal.
2. Route to the best chapter Skill when available.
3. Use the level ladder to choose C, B, A, or A_plus support.
4. Keep explanations course-aligned and age-appropriate.
5. Use common Skills only for generic fallback behavior.

## Assessment procedure

1. Select a level target.
2. Choose assessment style from the level ladder.
3. Ask short original questions.
4. Record gaps as learning barriers, not as fixed learner labels.

## Memory update guidance

Store only learning-support notes such as recurring gaps, preferred feedback style, and level-routing hints. Avoid permanent ability labels or private context that is not needed for future learning support.

## Comic lesson guidance

Use the original CathyGO comic worlds in `comic-bible.yaml`. Keep the beat order: hook, intuition, rule, guided_example, common_trap, student_check, recap, next_mission.

## Eval notes

Eval cases should cover course routing, level choice, learning-support memory, and chapter fallback behavior.
