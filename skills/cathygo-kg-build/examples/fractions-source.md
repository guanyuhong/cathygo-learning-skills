# Algebraic Fractions Mini Source

An algebraic fraction has a denominator that contains a variable. Before simplifying or solving, the original denominator cannot be zero.

Equivalent transformations may multiply or divide numerator and denominator by the same nonzero factor. A common mistake is canceling matching letters that are not complete factors.

When solving a fraction equation, candidate answers must be checked against the original denominators. A value that makes an original denominator zero is invalid even if it appears during algebraic solving.
