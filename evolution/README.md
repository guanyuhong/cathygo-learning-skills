# CathyGO Skill Evolution

CathyGO improves Learning Skills by learning from tutoring traces and converting failures into structured change proposals.

Proposals are the bridge between runtime learning and repository-governed Skills. Runtime systems can detect issues and suggest improvements, but official Skills are updated only through reviewed pull requests.

## Status levels

- `draft`: initial proposal or early-stage change.
- `experimental`: validated and minimally evaluated updates.
- `community`: reviewer-approved updates with broader eval coverage.
- `official`: maintainer-approved, fully governed, release-ready updates.
